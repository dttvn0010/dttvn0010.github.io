Tạo mới ứng dụng

  • Để tạo mới một ứng dụng Spring boot, có thể sử dụng công cụ https://start.spring.io/ theo hướng dẫn https://spring.io/guides/gs/spring-boot/

  • Ngoài cách trên, để tạo nhanh một ứng dụng, có thể theo các bước sau:
  • Tạo một folder cho ứng dụng, ví dụ demo
  • Trong folder của ứng dụng, tạo mới file pom.xml với nội dung như sau:

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
        xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 https://maven.apache.org/xsd/maven-4.0.0.xsd">
        <modelVersion>4.0.0</modelVersion>
        <parent>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-starter-parent</artifactId>
                <version>2.1.12.RELEASE</version>
                <relativePath/>
        </parent>
        <groupId>com.example</groupId>
        <artifactId>demo</artifactId>
        <version>1.0.0</version>
        <name>demo</name>

        <properties>
                <java.version>11</java.version>
                <maven.compiler.target>11</maven.compiler.target>
            <maven.compiler.source>11</maven.compiler.source>
        </properties>

        <dependencies>
                <dependency>
                        <groupId>org.springframework.boot</groupId>
                        <artifactId>spring-boot-starter-web</artifactId>
                </dependency>
        </dependencies>

        <build>
                <plugins>
                        <plugin>
                                <groupId>org.springframework.boot</groupId>
                                <artifactId>spring-boot-maven-plugin</artifactId>
                        </plugin>
                </plugins>
        </build>

</project>

  • Trong folder ứng dụng, tạo thư mục src để chứa source code ứng dụng. Trong thư mục src lần lượt tạo các thư mục con theo thứ tự src/main/java. Cấu trúc thư mục ứng dụng có dạng sau:

pom.xml

src

    └──main

        └──java

           

  • Trong thư mục java (thư mục trong cùng của cây thư mục source code), tạo mới file App.java :

package demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class App {
   
        public static void main(String[] args) {
                SpringApplication.run(App.class, args);
        }

}

  • Trong thư mục java tiếp tục tạo mới file SampleController.java với nội dung:

package demo;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;

@Controller
public class SampleController {

   @RequestMapping("/")
   @ResponseBody
   public String index() {
       return "Hello";
   }
}

  • Build ứng dụng : từ thư mục gốc ứng dụng (thư mục chứa file pom.xml), mở cửa sổ cmd và gõ lệnh:

        mvn install

        

  • Chạy ứng dụng: sau khi build xong ứng dụng, chạy ứng dụng bằng lệnh sau:

        mvn spring-boot:run

  • Kiểm tra ứng dụng sau khi chạy: Từ cửa sổ trình duyệt gõ địa chỉ http://localhost:8080 , màn hình trang chủ ứng dụng được tải lên có dạng như trong hình bên dưới:

Màn hình ứng dụng sau khi chạy

Xây dựng restful webservice:

  • Các loại restful service và mục đích sử dụng chính của mỗi loại:
  • GET: lấy dữ liệu
  • POST: tạo mới bản ghi
  • PUT: cập nhật bản ghi
  • DELETE: Xóa bản ghi

  • Trong spring boot, các restful service được tạo ra từ các RestController class. Các class này có khai báo @RestController ở trước khai báo class:        

                

@RestController

public class SampleController {

        ...

}

  • Khi có nhiều Controller, để tách biệt các restful service giữa các controller, thường mỗi controller sẽ được gắn một đường dẫn (route) riêng, bằng cách khai báo @RequestMapping ở trước khai báo class:

@RestController

@RequestMapping("/app1")

public class SampleController {

        ...

}

Khi đó, tất cả các restful service do controller cung cấp sẽ được gắn phần đường dẫn của controller vào phía trước đường dẫn của service.

Ví dụ:

package demo;

import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/app1")
public class SampleController {

   @GetMapping("/service1")
   public String service1() {
       return "Hello";
   }
}

Để truy cập đến restful service do hàm service1 cung cấp, cần dùng địa chỉ : http://localhost:8080/app1/service1

package demo;

import java.util.List;
import java.util.ArrayList;

import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/app1")
public class SampleController {
   
   static class Student {

       public int id;
       public String studentNo;
       public String name;
       
       public Student(int id, String studentNo, String name) {
           this.id = id;
           this.studentNo = studentNo;
           this.name = name;        
       }
   }
   
   @SuppressWarnings("serial")
   List<Student> students = new ArrayList<>() {{
       add(new Student(1, "10001", "Nguyen Van A"));
       add(new Student(2, "10002", "Nguyen Van B"));
   }};
 
   @GetMapping("/get_all_students")
   public List<Student> getAllStudents() {
       return students;
   }
   
   @PostMapping("/add_student")
   public Student addStudent(@RequestBody Student student) {
       var maxId = students.size() > 0? students.get(students.size() - 1).id : 0;
       student.id = maxId + 1;
       students.add(student);
       return student;
   }
}

package demo;

import java.util.List;
import java.util.ArrayList;

import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/app1")
public class SampleController {
   
   static class Student {

       public int id;
       public String studentNo;
       public String name;
       
       public Student(int id, String studentNo, String name) {
           this.id = id;
           this.studentNo = studentNo;
           this.name = name;        
       }
   }
   
   @SuppressWarnings("serial")
   List<Student> students = new ArrayList<>() {{
       add(new Student(1, "10001", "Nguyen Van A"));
       add(new Student(2, "10002", "Nguyen Van B"));
   }};
 
   @GetMapping("/get_all_students")
   public List<Student> getAllStudents() {
       return students;
   }
   
   @PostMapping("/add_student")
   public Student addStudent(@RequestBody Student student) {
       var maxId = students.size() > 0? students.get(students.size() - 1).id : 0;
       student.id = maxId + 1;
       students.add(student);
       return student;
   }
   
   @PutMapping("/update_student/{id}")
   public Student updateStudent(@PathVariable("id") int id, @RequestBody Student student) {
       student.id = id;
       for(int i = 0; i < students.size(); i++) {
           if(student.id == students.get(i).id) {
               students.set(i, student);
           }
       }
       return student;
   }
}

package demo;

import java.util.List;
import java.util.ArrayList;

import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/app1")
public class SampleController {
   
   static class Student {

       public int id;
       public String studentNo;
       public String name;
       
       public Student(int id, String studentNo, String name) {
           this.id = id;
           this.studentNo = studentNo;
           this.name = name;        
       }
   }
   
   @SuppressWarnings("serial")
   List<Student> students = new ArrayList<>() {{
       add(new Student(1, "10001", "Nguyen Van A"));
       add(new Student(2, "10002", "Nguyen Van B"));
   }};
 
   @GetMapping("/get_all_students")
   public List<Student> getAllStudents() {
       return students;
   }
   
   @PostMapping("/add_student")
   public Student addStudent(@RequestBody Student student) {
       var maxId = students.size() > 0? students.get(students.size() - 1).id : 0;
       student.id = maxId + 1;
       students.add(student);
       return student;
   }
   
   @PutMapping("/update_student/{id}")
   public Student updateStudent(@PathVariable("id") int id, @RequestBody Student student) {
       student.id = id;
       for(int i = 0; i < students.size(); i++) {
           if(student.id == students.get(i).id) {
               students.set(i, student);
           }
       }
       return student;
   }
   
   @DeleteMapping("/delete_student/{id}")
   public void deleteStudent(@PathVariable("id") int id) {
       var stOpt = students.stream().filter(st -> st.id == id).findFirst();
       stOpt.ifPresent(st -> students.remove(st));        
   }
}
